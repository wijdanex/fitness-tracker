# Fitness Tracker 

## Overview
Fitness Tracker is a web application built with **React** and **TailwindCSS** that allows users to log workouts, track progress, and explore exercises using live data from the **WGER API**.  
It helps users stay consistent with their fitness goals by providing workout history, progress charts, and exercise details.

## Features
- **Log Workouts**: Add exercises, sets, reps, and weights with timestamps.
- **Workout History**: View past workouts organized by date.
- **Progress Tracking**: Visualize progress with charts (total weight lifted, total workouts, etc.).
- **Exercise Library**: Fetch exercises from WGER API, search by name or muscle group.
- **Search Functionality**: Find exercises quickly with a responsive search bar.
- **Responsive Design**: Works on desktop, tablet, and mobile.

## Tech Stack
- **Frontend**: React (Vite), TailwindCSS
- **API**: WGER API (`https://wger.de/api/v2/exercise/`)
- **State Management**: React Hooks (`useState`, `useEffect`)
- **Deployment**: Netlify
